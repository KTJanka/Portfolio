CSS

As I mentioned in the HTML section with just HTML webpages are simply text based, the web would
be quite bland and boring with just HTML webpages. CSS (Cascading Style Sheets) comes in to fix
this issue by styling webpages for the user experience. CSS is able to add many features to a HTML
document, such as background colors/pictures, resizing content on the page and change the positioning
of the content. Every CSS file or element is a series of statements/instructions for the browser to
display the text with different styles. Statements identify the element it needs to style and tells
the browser how to do draw the styles. Selectors are used for identifying the HTML tag that needs
styled. The selector chooses a tag to style with curly brackets following the tag, the style
properties will be inside the brackets, the properties are given a value and ended with a semicolon.
