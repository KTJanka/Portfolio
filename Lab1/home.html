HTML

HTML (Hyper Text Markup Language) is a computer language that is the foundation of all webpages.
It is a text based language, with only HTML most webpages are very basic lines of text.
Hypetext is the method that you move around online using special text called hyperlinks, hyper meaning that it is linear
or that there is no order on how you move around the web. Markup is done with HTML tags changing how text works
(bold,italicised,etc...). Tags are what seperate simple text from HTML text. HTML has gone through man versions through
the years:

  • HTML 1.0 - The first release, not many tools other than getting basic text on the web.
  • HTML 2.0 - Was the standard until Jan. 1997, defined many core features of HTML for the first time.
  • HTML 3.0 - Meant to include many new features, but browsers were to slow. 3.0 spec was eventually abandoned altogether.
  • HTML 3.2 - Designed by W3C (World Wide Web Consortium) became standard in Jan. `97.
  • HTML 4.01 - Large update of HTML standards and last iteration of the classic HTML. Official standard April 1998.
  • XHTML 1.0 - Basically it was a new set of coding rules using XML. Was joint standard with 4.01 since Jan 26, 2000.
  • HTML 5 - Designed for the web, not only now, but for the future. (I will cover this in detail in the HTML5 section).
