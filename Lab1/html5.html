HTML5

When HTML was first implemented the amount of users on the web was drastically lower than
todays standards. With the sudden change of technology over the years (smartphones, tablets etc...)
many new challenges came for developers and engineers. To address these new challenges HTML5 was
introduced to provide smoother and more consistent experiences for web designers and users alike.
HTML5 was a collaboration between W3C and WHATWG (Web Hypertext Application Technology Working Group).
They started working on HTML5 in 2006 to address reliance on plugins and error handling issues with
HTML. They added more markups and greatly simplified the process of creating web applications.
Some great features include built in video and audio capabilities, as well as the ability to store
user data locally rather than using cookies. With the constant updating of HTML5 developers are having
a more consistent experince when it comes to error handling of the code.
